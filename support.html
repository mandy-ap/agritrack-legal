Questions fréquentes

Comment sauvegarder mes données ?

Allez dans Paramètres → Sauvegarde et restauration → Exporter. Un fichier .agritrack sera généré et vous pourrez le partager vers iCloud, OneDrive, ou tout autre emplacement de votre choix.

Comment restaurer une sauvegarde ?

Dans Paramètres → Sauvegarde et restauration → Restaurer, sélectionnez votre fichier .agritrack. Attention : la restauration remplace toutes les données actuelles.

Je ne reçois pas les notifications de couvée.

Vérifiez que les notifications sont autorisées pour AgriTrack Mini dans les réglages de votre appareil (Réglages → Notifications → AgriTrack Mini). Sur Android, assurez-vous également que l'optimisation de batterie n'est pas activée pour l'app.

Comment changer la langue de l'application ?

Allez dans Paramètres → Ma ferme → Langue et sélectionnez FR ou EN.

J'ai perdu mes données après une réinstallation.

Les données sont stockées localement sur votre appareil et ne sont pas récupérables après une désinstallation sans sauvegarde préalable. Nous recommandons d'activer les sauvegardes automatiques dans les paramètres.

Comment supprimer toutes mes données ?

Dans Paramètres → Zone de danger → Supprimer toutes les données. Cette action est irréversible.

Confidentialité

AgriTrack Mini ne collecte aucune donnée personnelle et ne se connecte à aucun serveur. Consultez notre politique de confidentialité complète.
